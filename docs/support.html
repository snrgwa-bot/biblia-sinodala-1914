Intrebari Frecvente

Cum activez functiile AI?

Mergeti la Setari > introduceti cheia API Gemini (obtineti una gratuita de pe ai.google.dev), apoi abonati-va la planul Premium.

Cum anulez abonamentul?

Mergeti la Setari iPhone > Apple ID > Abonamente > Biblia Sinodala > Anuleaza.

Datele mele sunt stocate in cloud?

Nu. Notele si evidentierile sunt stocate doar pe dispozitivul dumneavoastra.

Pot folosi aplicatia offline?

Da. Textul Bibliei si enciclopedia functioneaza offline. Doar functiile AI necesita conexiune la internet.

Ce editie a Bibliei este folosita?

Biblia Ortodoxa Sinodala, Editia Sfantului Sinod, 1914 — text din domeniul public disponibil pe archive.org.

Aplicatia functioneaza pe iPad?

Da. Aplicatia suporta iPad cu layout pe doua coloane pentru o experienta de citire superioara.
